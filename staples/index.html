Feature Crop

The FeatureCrop transformation tries to determine the most important features of the input image and performs a crop around as many of these features as possible.
FeatureCrop works very similarly to RegionOfInterestCrop except for the region of interest is specified automatically by the found features instead of manually.

Feature crop (300x300): IMQuery FeatureCrop,width=300,height=300

Resize

Resize option.
The Resize transformation is used to resize (change the dimensions of) images. In the exaple only width parameter being used and we keep the aspect ratio.

Resize (100) IMQuery im=Resize=100"

Crop

Crop option.
The Crop transformation is used to crop and/or expand the canvas area of images.

Resize (100) IMQuery im=Resize=100"
